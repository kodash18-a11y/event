# Стефорд — GitHub Pages пакет

Внутри уже готовые файлы для публикации на GitHub Pages.

## Что загружать в репозиторий
- `index.html`
- папку `assets`
- файл `.nojekyll`

## Как опубликовать
1. Создайте новый репозиторий на GitHub.
2. Загрузите туда `index.html`, папку `assets` и `.nojekyll`.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/ (root)`.
6. Сохраните настройки.
7. Через 1–3 минуты GitHub выдаст ссылку на сайт.
